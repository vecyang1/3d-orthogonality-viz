# 波的正交性 3D 互动演示 (Orthogonality Visualization)

这是一个基于 Three.js 开发的互动演示，旨在可视化数学中“正交函数”的概念。

## 核心数学概念

### 1. 投影与轴 (Projection & Axes)
在 2D 空间中，我们用 $x$ 和 $y$ 轴作为基向量。在信号处理（频率分析）中，我们可以将不同的正弦波频率视为不同的“轴”。
- **X-Z 平面 (蓝色波形)**: 代表频率 $f_1$。
- **Y-Z 平面 (粉色波形)**: 代表频率 $f_2$。
- **3D 路径 (白色波形)**: 代表两个频率的叠加效果。

### 2. 正交验证 (Orthogonality)
为什么 $f_1$ 的投影里没有 $f_2$ 的成分？因为它们是正交的。
本项目通过计算内积（即乘积的积分）实时展示：
$$\int_{0}^{T} \sin(2\pi f_1 t) \cdot \sin(2\pi f_2 t) dt \approx 0 \quad (f_1 \neq f_2)$$
当频率不同且在一个周期内观察时，它们的投影互不干涉。

## 技术栈
- **Vite**: 前端构建工具。
- **Three.js**: 3D 渲染引擎。
- **CSS**: Vanilla CSS (Glassmorphism 风格)。

## 如何运行
1. 安装依赖: `npm install`
2. 启动开发服务器: `npm run dev`
3. 访问浏览器中的 Local URL (通常是 http://localhost:5173)。

## 操作指南
- 调节频率滑块，观察 3D 路径如何变化。
- 旋转和缩放 3D 场景，查看 XZ 和 YZ 平面上的“投影”
- 观察左下角的数学模型验证。
